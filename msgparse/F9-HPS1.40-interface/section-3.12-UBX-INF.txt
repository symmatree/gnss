3.12 UBX-INF (0x04)
Messages in the UBX-INF class are used to output strings from the firmware or application code. All
messages have an associated type to indicate the nature or priority of the message.
3.12.1 UBX-INF-DEBUG (0x04 0x04)
3.12.1.1 ASCII output with debug contents
Message UBX-INF-DEBUG
ASCII output with debug contents
Type Output
Comment This message has a variable length payload, representing an ASCII string.
Header Class ID Length (Bytes) Payload Checksum Message
structure 0xb5 0x62 0x04 0x04 [0..n] see below CK_A CK_B
Payload description:
Byte offset Type Name Scale Unit Description
Start of repeated group (N times)
0 + n CH str - - ASCII Character
End of repeated group (N times)
